Which Programming Language is Right for You?

This page will help you find a programming language that suits your personality and style.

Note: There are many languages out there with similar properties, but it is still worth choosing a specific language to start on and build from.

C

C is the OG programming language. First released in 1972 (That's 48 years ago!), this programming language has seen it all and stuck around.

This is a difficult language to start with for beginners, which will likely mean more time spent figuring out how the language works rather than being able to spend time learning the fundamentals of programming.

Learning this language is not going to be easy as the syntax is fairly dated, but past that should be pretty straightforward. Plus, learning this language will be a good foundation and base for learning other programming languages. Because it has been around since the beginning, its influence on other programming languages will be noticeable wherever you decide to go next. This aint yo mama's programming language (More like yo grandma's programming language).

C++

C++ is a powerful language that can do many things, but is rather low level, meaning it is more difficult to "read." This means you will spend more time figuring out how the language itself works rather than how programming works.

Writing an application from scratch with C++ require a lot of code and is more difficult to manage. However, C++ is good at what it does and is often used for very resource intensive programs like AAA games. C++ is a good choice for computer software, mobile apps, video games, game engines, or even web applications.

C#

C# (pronounced C-sharp) was developed by Microsoft for their .NET framework and is based on C and C++. It was designed to be simple and straightforward, which makes it an easier language to start on.

A high-level language (though not quite as high-level as, say, Python), it is designed to be easy to read, and the language itself takes care of a lot of the abstracts behind the scenes so your computer can deal with the code.

C# is the primary programming language of Microsoft and is also popular with game developers, so if you are interested in working for/with Microsoft, or getting into game development to use things like the Unity engine, this may be the language for you!

CSS

CSS (which stands for Cascading Styles Sheets) is the language many web designers and web developers use for the look and design of their websites. If HTML is the foundation and framework of a house, CSS is the siding, style, and paintjob. Together with javascript, these 3 language form the trinity of web development.

CSS is not considered by many to be a "proper" coding language, but nonetheless is essential in web development. If you are interested in building websites, CSS will be required. Luckily, CSS is a quick and easy language to learn the basics of, and is great for beginners because you can see your work immediately.

Another thing that makes CSS great is that you can learn enough in a few weeks to start picking up side jobs to practice and earn a little money on the side while you continue learning other languages, which will help pay the bills as well as deepen your portfolio for landing that entry-level front-end developer job down the road.

Python

Python is one of the most beginner friendly programming languages available and is great for learning the fundamentals of coding. It is a high-level coding language, meaning it reads almost like English, so it is very intuitive and clean and decreases the amount of time you have to spend learning how the language itself works.

Python can be used for some very cool and versatile things, such as data mining, automation, machine learning, web applications, and game development.

Python is a very popular language and has many resources available to individuals interested in learning, and there is a strong community around Python that is ready and willing to help aspiring Python programmers.

Ruby

Ruby is a very high-level programming language that has been around for 25 years. It is a creative programming language that often allows the user to find multiple solutions to a given problem. Using Ruby, you can see results fairly quickly and build sites surprisingly quickly, especially using something like the Ruby on Rails framework. This makes Ruby the most useful web development programming language for small businesses and startups to launch web applications as quickly as possible.

If small tech companies or startups are what you are interested in, Ruby is a good choice for a starter language. Just be sure that the companies you apply to are using Ruby.

Java

Java is one of the most widely used and most popular programming languages in the world right now. It can be used for many types of applications, from mobile apps to entire desktop applications. As of June 2020, it had fallen to second place to C in the TIOBE Index.

Many aspects of Java come from C++, but it is a bit easier to learn for beginners. One selling point of Java is that it is designed to allow developers to write code that will work on entirely different devices, hence their slogan "write once, run anywhere."

Java is very popular for many types of applications, which makes Java one of the most in demand and highest paying programming languages you can learn. It is also one of the most useful programming languages for Android mobile development, so if you are interested in writing apps for Android phones, this is your ticket in.

Because Java is very versatile, it makes a great language to learn if you are not quite sure what you want to do with programming.It is a great asset for applying for initial coding jobs. While it is very useful and very versatile, it is not the easiest or most beginner friendly language out there (think Python or Ruby for that).

Javascript

JavaScript is not the same thing as Java. JavaScript is used together with HTML to make websites interactive, interesting, and user-friendly. Nearly every webpage you navigate to uses JavaScript. It is one of the most popular web development programming languages out there and, consequently, is a good language to have on a resume.

Thera are multiple different frameworks and libraries (pre-made coding materials that the coder can just draw from for ease of use), including Vue, React, Angular, Ember, and Node. Each of these frameworks brings similar designs to the language, but in different ways. The introduction of Node allows the developer to create client side content as well, making JavaScript a full-stack language

If you want to develop websites, you have to learn JavaScript. But pair with HTML and CSS and you have the skills to create quite the web developer portfolio and land jobs in web development. And because JavaScript is so popular right now, you won't have any trouble landing those jobs.

HTML

HTML (or Hypertext Markup Language) is used to create all websites and is the essential structural and content component of said sites. While HTML is also not technically a full "programming language," but a markup language, HTML is arguably the most important part of web development. The basics are very simple to learn and it is a great jumpoff point to coding and web development.Together with CSS and Javascript, you will be able to create some pretty amazing website.

Perl

Perl is a very powerful and very versatile programming language. Originally coming onto the scene in 1987, this language has been around for a while and has since split into two separate families of the language, Perl 5 and Perl 6. Perl was originally designed for text manipulation and processing, but has since staked ground in everything from web development to system administration. Perl can also be used on nearly every computer system on the planet, earning it the title "Swiss army knife of programming langauges."

Perl can be compared to multiple different languages for multiple different things. Many features are similar to C, Python, and PHP. However, it allows multiple solutions to a given problem unlike Python and PHP, and more similar to Ruby.

Perl also can seem a bit jumbled or crowded to a beginner programmer, but once you get the hang of it, it becomes much easier to learn.

Perl is also not as popular as many other languages on this list, and therefore if you choose Perl, you should make sure the company you are applying to actually uses Perl.

PHP

PHP is a scripting language working on the server side of web development projects written in HTML. It is relatively easy to learn, making it one of the most popular programming languages available and consistently landing it in the top 10 on the TIOBE Index. It is used by 80% of the top 10 million websites online, showing just how widely used it is.

Because PHP is a server side language, like Pyhton and Perl, it can be used to create things like login pages, photo galleries, discussion forums, etc. Some of the most popular websites in the world use PHP, such as Facebook, Tumblr, and Wordpress.

If you are looking into a career in web development, you can't do wrong choosing PHP. It is the most useful language for WordPress development and is a very safe choice if you plan to do freelance web development.
